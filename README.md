# CountryJoinMessage

Changes the Bukkit/Spigot join message to: 'Player joined from [Country]'.

This plugin:
 * Is lightweight
 * Does **not** track you and **no** statistics are collected
 * Works offline
 * [Depends on GeoIPTools2!](https://dev.bukkit.org/projects/sindastras-legacy-geoiptools)
